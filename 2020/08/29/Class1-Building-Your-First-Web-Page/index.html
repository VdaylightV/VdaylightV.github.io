#Class1: Building Your First Web Page

1.Difference between HTML & CSS

  • Generally, the HTML & CSS are independent of each other while they need to combine and cooperate with each other to construct a pretty good website.
  • To be more specific, the HTML defines the structure and the content of the website. Meanwhile, the CSS determines the style and appearance of the website.

2.Terms of HTML

Elements

The examples above are some elements of the website. Therefore, elements are designators that define the structure and content of objects within a page.

Tags

The examples above are tags which mark the beginning and end of the elements respectively. The content between the beginning mark and the end mark of the element is the content of the element.

Attributes

The code above is an example of the attributes, which provides a hyperlink reference to a linked resource. In conclusion, attributes are properties used to provide additional information about an element.
