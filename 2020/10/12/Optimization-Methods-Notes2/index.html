[Optimization Methods]——Notes 2

保凸运算

  • 交集：$S_1$ and $S_2$ are convex, then $S_1 \cap S_2$ is convex.

可拓展到多个：$\forall \alpha \in \cal A$,$S_{\alpha}$ is convex, then $\cap_{\alpha\in\cal A}S_{\alpha}$ is convex.

  • 子空间、仿射集合、凸锥的交集都是封闭的
  • 多面体是半空间和超平面的交集——是凸的
  • 半正定锥可表示为多个半空间的交集——是凸的

仿射函数

  • 仿射函数：函数$f:{\R}^n \to {\R}^m$是仿射的$\iff\iff$它是一个线性函数和常数的和

    若$S\subset {\R}^n$且 $S$是凸的，那么$S$在$f$下的象：$f(S)=\{f(x)|x\in S\}$是凸的

    $S$在$f$下的原象:$f^{-1}(S)=\{x|f(x)\in S\}$也是凸的

    • 伸缩

      $\alpha S = \{\alpha x| x\in S\}$

      • 若$S$为凸的，则$\alpha S$也是凸的

    • 平移

      $S+\alpha = \{x+\alpha|x\in S\}$

      • 若$S$为凸的，则$S+\alpha$也是凸的

    • 投影

      若$S\subset {\R}^m×{\R}^n$是凸集，那么$T=\{x_1\in {\R}^m|(x_1,x_2)\in S对于某些x_2\in{\R}^n\}$也是凸集

    • 两个集合的和

      $S_1+S_2=\{x+y|x\in S_1,y\in S_2\}$

      若$S_1,S_2$是凸的，则$S_1+S_2$也是凸的

    • 两个集合的直积或Cartesian乘积

      $S_1×S_2=\{(x_1,x_2)|x_1\in S_1,x_2\in S_2\}$

      若$S_1,S_2$是凸的，则$S_1×S_2$也是凸的

      将线性函数$f(x_1,x_2)$作用于$S_1×S_2$,则可得到$S_1+S_2$

    • 两个集合的部分和

      $S_1,S_2\in {\R}^n×{\R}^m$,$S=\{(x,y_1+y_2)|(x,y_1)\in S_1, (x,y_2)\in S_2\}$

      若$S_1,S_2$是凸的，那么$S$也是凸的

      m = 0:两个集合的交集

    n = 0: 两个集合的和

    • 多面体

      • 非负象限${\R}^n_+$：具有非负分量的点的集合$\to$是多面体也是锥

      多面体:$\{x|Ax\preceq b, Cx = d\}$可以表示为非负象限和原点的Cartesian乘积在仿射函数$f(x)=(b-Ax,d-Cx)$下的原象

      $\{x|Ax\preceq b, Cx = d\} = \{x|f(x)\in {\R}^m_+ ×\{0\}\}$

    • 线性矩阵不等式(LMI)

      $A(x)=x_1A_1+…+x_nA_n \preceq B,A_i,B\in S^m$

      其解为：$\{x|A(x) \preceq B\}\to$是凸集

      是半正定锥在由$f(x)=B-Ax$给定的仿射映射$f:{\R}^n\to{S}^m$下的原象

    透视函数和线性分式函数

    • 透视函数：$P:{\R}^{n+1}\to {\R}^n\iff P(z,t) = \frac{z}{t}$,dom $P={\R}^n×{\R}_{++}$

      透视函数对向量进行伸缩或规范化——将其最后一维分量置为1并舍弃

      具有保凸性：

      • 若$C\in$ dom P是凸的，其象$P(C)=\{P(x)|x\in C\}$也是凸的
      • 若$C\in {\R}^n$是凸的，其逆象$P^{-1}(C)=\{(x,t)\in {\R}^{n+1}|\frac{x}{t}\in C, t \ge 0\}$

      小孔成像原理——通过小孔看凸的物体还是凸的

    • 线性分式函数：

      函数$g：{\R}^n\to{\R}^{m+1}$是仿射的

      ​ $g(x) = \left[ \begin{matrix} A \ c^T \end{matrix} \right]x + \left[ \begin{matrix} b\ d \end{matrix} \right]$

      ​ $A\in {\R}^{m×n},b\in{\R}^m,c\in{\R}^n,d\in{\R}$,

​ 函数$f = P\circ g,f:{\R}^n\to {\R}^m\iff$ $f(x)=(Ax+b)/(c^Tx+d)$, dom $f = {x|c^Tx+d>0}$，成为线性分式（或投射）函数

由透视函数和仿射函数复合而成

若$c=0,d>0$，则$f$的定义域为${\R}^n$，$f$是仿射函数——仿射和线性函数为特殊的线性分式函数

利用矩阵表示:

用Q左乘$(x,1)$,即可得到$(Ax+b,c^Tx+d)$,继而用透视函数$P$作用于之上,即可得到$(f(x),1)$,$f(x) = \frac{Ax+b}{c^Tx+d}$

 具有保凸性:
  • 若$C$是凸的且$\{c^Tx+d>0 ~for ~x \in C\}$,则$f(C)=\{\frac{Ax+b}{c^Tx+d}|x\in C\}$是凸的
  • 若$C\in {\R}^m是凸的,则其逆象f^{-1}(C)=\{x|\frac{Ax+b}{c^Tx+d}\in C\}$是凸的

广义不等式

  • 正常锥:锥$K\subset {\R^n}$为正常锥,若$K$满足以下条件

    • $K$是凸的
    • $K$是闭的
    • $K$是实心的,内部非空
    • $K$是尖的$\iff$单向的,不包含直线$\iff$$x\in K,-x\in K \Rightarrow x=0$

    利用锥$K$来定义广义不等式: $x \preceq_K y \iff y-x \in K$

    • 严格偏序关系: $x \prec_K y \iff y-x \in$ int K

    并不是任意两个元素都可比,存在两个元素之前不存在大小关系的情况

  • 非负象限和分量不等式:

    • 正常锥$K={\R}^n_+$
    • $x \preceq_K y \iff x_i \le y_i, i = 1,…,n$
    • $x \prec_K y \iff x_i < y_i, i = 1,…,n$

  • 半正定锥和矩阵不等式

    • 正常锥$K = {S}^n_+$
    • $X \preceq_K Y \iff Y-X ~is~PSD $
    • $X \prec_K Y \iff Y-X ~is~positive ~definite $

  • 广义不等式的性质

    • 加法保序性
    • 传递信
    • 非负数乘保序性
    • 自反性
    • 反对称性
    • 极限运算保序性

    严格广义不等式的性质:

    • 满足严格一定满足广义
    • 严格+广义$\to$严格
    • 严格+数乘($\alpha>0$)$\to$严格
    • $x \nprec_k x$
    • 若$x\prec_k y$,对于足够小的$u,v$有$x+u\prec_ky+v$

  • 最小元:$x\in S$ is the minimum element$\iff$ $\forall y \in S, 满足x\preceq_k y$

    $S\subset x+ K$

    若最小元存在,则其唯一

  • 极小元:$x\in S $ is a minimal element $\iff$ if $y\in S,y\preceq_K x$ only if $y = x$

    $(x-K)\cap S = \{x\}$

    极小元可能有多个

  • 同理可定义极大元和最大元
