[Optimization Methods]——Notes 1

数学补充——分析

  • 内部：对于$x\in C\subset \R^n$,若存在$\epsilon>0$满足:$\{y|~||y-x||_2 \le \epsilon\}\subset C$,称$x$为$C$的内点，$C$的所有内点组成$C$的内部，记为int C
  • 开集：若int C = C,则称C为开集
  • 闭集：若$C\subset \R^n$的补集$R^n$ $C$ = $\{x\in\R|x\notin C\}$是开集，则称$C$为闭集
  • 闭包：集合$C$的闭包定义为：cl C = $\R^n$ int($\R^n$ C)——$C$的补集的内部的补集

点$x$属于$C$的条件为：对与每一个$\epsilon>0, \exist y \in C$满足$||x-y||_2 \le \epsilon$

利用收敛序列和极限点的观点：

  • 集合$C$是闭集$\iff$它包含它的每个收敛序列的极限点
  • $C$的闭包是$C$的所有收敛序列的极限点组成的集合
  • 边界：bd C = cl C int C

边界点$x$的性质：

  • 对所有$\epsilon > 0$,存在$y\in C$和$z\notin C$满足$||y-x||_2\le\epsilon,||z-x||_2\le \epsilon$

集合$C$是闭集的条件为：bd C $\subset C$

集合$C$是开集的条件为：$C \bigcap$ bd C = $\emptyset$

仿射集合

  • 若通过集合C中的任意两点的直线仍然在集合C中，则称集合C是仿射的。

仿射组合：$\theta_1+…+\theta_k = 1$,$\theta_1x_1+…+\theta_kx_k$称为$x_1…x_k$的仿射组合。

线性方程组的解集是一个仿射集合，$C=\{x|Ax=b\}$,与仿射集合C相关联的子空间就是A的零空间

  • 零空间：算子$A$的零空间是$Av = 0$的所有解$v$的集合
  • 子空间：C是仿射集合且$x_0\in C$,则集合$V=C-x_0=\{x-x_0|x\in C\}$,是一个子空间，关于加法和数乘封闭。

同理：仿射集合C可以表示为：$C=V+x_0=\{v+x_0|v\in V\}$

  • 仿射包：集合$C\subset R^n$中的点的所有仿射组合组成的集合称为$C$的仿射包，即为aff C

aff C = $\{\theta_1x_1+…+\theta_kx_k|x_1,…,x_k \in C, \theta_1+…+\theta_k=1\}$

  • 仿射包是包含$C$的最小仿射集合。
  • 相对内部：集合$C$的相对内部为它相对于aff C的内部，记为relint C，即：relint C = $\{x\in C|B(x,r)\bigcap$aff C $\subset C$ 对于某些$r>0\}$
  • 相对边界：cl C relint C——集合$C$的闭包除去集合$C$的相对边界

凸集

  • 凸集：集合$C$被称为凸集$\iff\forall x_1,x_2\in C$和满足$0\le \theta\le1$的$\theta$都有$\theta x_1+(1-\theta)x_2 \in C$

仿射集合一定是凸集

凸组合：$\theta_1x_1+…+\theta_kx_k$,为$x_1,…,x_k$的一个凸组合，其中$\theta_1+…+\theta_k=1$且$\theta_i \ge 0, i = 1,…,k$

一个集合是凸集$\iff$集合包含其中所有点的凸组合

  • 凸包：集合$C$中所有的的凸组合的集合称为其凸包，记为conv C，

conv C = $\{\theta_1x_1+…+\theta_kx_k|x_i\in C,\theta_i\ge0, i = 1,…,k,\theta_1+…+\theta_k = 1\}$

凸包conv C总是凸的，它是包含$C$的最小凸集

锥

  • 锥：若对任意$x\in C$和$\theta \ge 0$, 都有$\theta x \in C$,则将集合$C$称为锥或非负其次。
  • 凸锥：若$C$是锥且$C$是凸的，则称$C$为凸锥$\iff$ $\forall x_1,x_2\in C和\theta_1,\theta_2 \ge0,有\theta_1x_1+\theta_2x_2 \in C$

锥组合（非负线性组合）：$\theta_1x_1+…+\theta_kx_k,\theta_1,…,\theta_k \ge 0$形式的点称为$x_1,…,x_k$的锥组合

集合$C$是凸锥$\iff$它包含其元素的所有锥组合

  • 锥包：集合$C$的所有锥组合的集合：$\{\theta_1x_1+…+\theta_kx_k | x_i \in C, \theta_i \ge 0, i = 1,…,k\}$

集合$C$的锥包是集合$C$的最小的凸锥

凸集的例子

  • $\emptyset$、单点集$\{x_0\}$、全空间$\R^n$
  • 直线是仿射的，通过零点的直线是凸锥
  • 线段是凸的但不是仿射的
  • 射线：$\{x_0+\theta v|\theta \ge 0\}$ ，$v\ne 0$ 的集合是凸的，但不仿射；若$v=0$,则其是凸锥
  • 任意子空间是仿射的、凸锥

  • 群：一个集合和定义在该集合上的操作的组合若满足下列条件则称为群

    • 在该操作上的封闭性

    • 操作的结合律

    • 集合中存在不变元

    • 集合中存在逆元

      若额外满足操作的交换律则称为Abelian Group

  • 向量空间：一个集合、定义在集合内部元素之间的操作和与定义在集合中元素与外部集合元素之间的操作组合（e.g. 内部：加法；外部：数乘）

    • 除去外部操作的得到的群是Abelian Group

    • 对于外部操作满足分配律和结合律

    • 对于外部操作也存在不变元

  • 向量子空间：$V = (\cal V,+,·)$是向量空间，集合$\cal U \subset \cal V$且$\cal U \ne \emptyset$，$U=(\cal U,+,·)$称为$V$的向量子空间,记为$U\subset V$,如果以下条件满足：

    • $U$关于$\cal U ×\cal U$、$\R × \cal U$封闭
    • $0 \in \cal U$

  • 关于子空间和向量空间的一些性质：

    • 对任一向量空间$V$，其自身$V$和$\{0\}$都是其子空间
    • 子空间至少要包含零元($0$)

超平面和半空间

  • 超平面：具有如下形式的集合$\{x|a^Tx=b\}$,$a\in\R^n,a\ne 0,b\in \R$ $\iff$ $\{x|a^T(x-x_0)=0\} = x_0+a^{\bot}$,$x_0$为超平面上任意一点

$a^{\bot}$表示$a$的正交补，即与$a$正交的向量的集合：$a^{\bot}=\{v|a^Tv=0\}$

超平面由偏移$x_0$加上所有正交于法向量$a$的向量构成。

  • 半空间：一个超平面将$\R^n$划分为两个半空间：$\{x|a^Tx\le b\},a\ne 0\iff \{x|a^T(x-x_0)\le 0\}$,$x_0$是超平面上的任意一点。

半空间是凸的但不是仿射的

  • 若半空间为$\{x|a^Tx\le b\}$
    • 半空间的边界——超平面$\{x|a^Tx=b\}$
    • 半空间的内部——开半空间$\{x|a^Tx<b\}$

球和椭球

  • 球：

    $B(x_c,r)=\{x|~||x-x_c||_2\le r^2\} = \{x|(x-x_c)^T(x-x_c)\le r^2\} = \{x_c+ru~|~||u||_2\le 1\}$

$向量x_c$是球心，$标量r$是半径，$B(x_c,r)$表示所有距离$x_c$不超过$r$的所有点的集合

球是凸集

  • 椭球：

​ $\varepsilon = \{x|(x-x_c)^TP^{-1}(x-x_c)\le1\} = \{x_c+Au~|~||u||_2 \le 1\}$

$x_c\in \R^n$是椭球中心，$P=P^T\succ 0$,即$P$是对称正定矩阵——决定了椭球从$x_c$向各个方向的拓展幅度,半轴长为$\sqrt\lambda_i$

$\varepsilon = \{x_c+Au~|~||u||_2 \le 1\}$，其中可以假设$A=P^{\frac{1}{2}}$是对称正定的，当$A$为对称半正定矩阵，但奇异时，集合称为退化的椭球，仿射维数为$A$的秩。

退化的椭球也是凸的

范数球和范数锥

  • 范数球：$\{x~|~||x-x_c|| \le r\}$

    范数球是凸的

  • 范数锥：$C=\{(x,t)~|~||x||\le t $ \} $\subset \R^{n+1}$

    范数锥是一个凸锥

    • 二阶锥：

    $C=\{(x,t)\in \R^{n+1}|~||x||_2 \le t$}$\iff x^2-t^2 \le 0, t \ge 0$

多面体和单纯形

  • 多面体：$\cal P = \{x|a_j^Tx\le b_j\, j = 1,…,m,c_j^Tx=d_j,j=1,…,p\}$

    多面体对应有限个线性不等式和等式的解集

    一系列半空间和超平面的交集

    • 矩阵表示：

      $\cal P = \{x|Ax \preceq b, Cx = d\}$

    • $u \preceq v \iff \forall i,~ u_i \preceq v_i $

  • 单纯形：设$k+1$个点$v_0,….,v_k\in \R^n$仿射独立，那么这些点决定了一个单纯形：

    C = conv $\{v_0,…,v_k\}$=$\{\theta_0v_0+…+\theta_kv_k~|~\theta \succeq 0, 1^T\theta = 1\}$

    • 仿射独立：$k+1$个点$v_0,…,v_k\in \R^n$仿射独立，即，$v_1-v_0,…,v_k-v_0$线性独立

    单纯形是一类重要的多面体。

    • 上述单纯形的仿射维数为$k$,也成为$\R^n$空间的$k$维单纯形。

  • 常见的单纯形：

    • 一维单纯形——线段

    • 二维单纯形——三角形及其内部

    • 三维单纯形——四面体

    • 单位单纯形：零向量$0$和单位向量$\epsilon_1,…\epsilon_n \in \R^n$决定的$n$维单纯形

      $x\succeq0, 1^Tx \le 1$

    • 概率单纯形：单位向量$\epsilon_1,…\epsilon_n \in \R^n$决定的$n-1$维单纯形

      $x\succeq0,1^Tx=1$

      对应$n$个元素的集合的概率分布，$x_i$可理解为第$i$个元素的概率
