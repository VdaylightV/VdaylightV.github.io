计算机视觉历史回顾与介绍

Visual Data

  • Most are in the form of pixels.
  • The visual data are ‘Dark Matter’ in the Internet.
  • To label, classify and index the pictures or every frame of a video.

History

  • The begining of visual processing is not the whole stuff, but the simple structures, for example edge. (1950s-1960s)

  • Lary Roberts‘s exploration on Block World, the outcome is that the edge defines the stuff. (1963)

  • David Marr assets that vision is hieraechical. The first one is edge layer, which represents edge. The second layer is 2.5D, which construct 2D pictures to 3D.

  • Cylinder Model uses simple shapes to represents all the stuff in the world.

  • Pictorial Model still uses some simple structures to represents the stuff.
