[Optimization Methods]——Notes 2

保凸运算

  • 交集：$S_1$ and $S_2$ are convex, then $S_1 \cap S_2$ is convex.

可拓展到多个：$\forall \alpha \in \cal A$,$S_{\alpha}$ is convex, then $\cap_{\alpha\in\cal A}S_{\alpha}$ is convex.

  • 子空间、仿射集合、凸锥的交集都是封闭的
  • 多面体是半空间和超平面的交集——是凸的
  • 半正定锥可表示为多个半空间的交集——是凸的

仿射函数

  • 仿射函数：函数$f:{\R}^n \to {\R}^m$是仿射的$\iff\iff$它是一个线性函数和常数的和

    若$S\subset {\R}^n$且 $S$是凸的，那么$S$在$f$下的象：$f(S)=\{f(x)|x\in S\}$是凸的

    $S$在$f$下的原象:$f^{-1}(S)=\{x|f(x)\in S\}$也是凸的

    • 伸缩

      $\alpha S = \{\alpha x| x\in S\}$

      • 若$S$为凸的，则$\alpha S$也是凸的

    • 平移

      $S+\alpha = \{x+\alpha|x\in S\}$

      • 若$S$为凸的，则$S+\alpha$也是凸的

    • 投影

      若$S\subset {\R}^m×{\R}^n$是凸集，那么$T=\{x_1\in {\R}^m|(x_1,x_2)\in S对于某些x_2\in{\R}^n\}$也是凸集

    • 两个集合的和

      $S_1+S_2=\{x+y|x\in S_1,y\in S_2\}$

      若$S_1,S_2$是凸的，则$S_1+S_2$也是凸的

    • 两个集合的直积或Cartesian乘积

      $S_1×S_2=\{(x_1,x_2)|x_1\in S_1,x_2\in S_2\}$

      若$S_1,S_2$是凸的，则$S_1×S_2$也是凸的

      将线性函数$f(x_1,x_2)$作用于$S_1×S_2$,则可得到$S_1+S_2$

    • 两个集合的部分和

      $S_1,S_2\in {\R}^n×{\R}^m$,$S=\{(x,y_1+y_2)|(x,y_1)\in S_1, (x,y_2)\in S_2\}$

      若$S_1,S_2$是凸的，那么$S$也是凸的

      m = 0:两个集合的交集

    n = 0: 两个集合的和

    • 多面体

      • 非负象限${\R}^n_+$：具有非负分量的点的集合$\to$是多面体也是锥

      多面体:$\{x|Ax\preceq b, Cx = d\}$可以表示为非负象限和原点的Cartesian乘积在仿射函数$f(x)=(b-Ax,d-Cx)$下的原象

      $\{x|Ax\preceq b, Cx = d\} = \{x|f(x)\in {\R}^m_+ ×\{0\}\}$

    • 线性矩阵不等式(LMI)

      $A(x)=x_1A_1+…+x_nA_n \preceq B,A_i,B\in S^m$

      其解为：$\{x|A(x) \preceq B\}\to$是凸集

      是半正定锥在由$f(x)=B-Ax$给定的仿射映射$f:{\R}^n\to{S}^m$下的原象

    透视函数和线性分式函数

    • 透视函数：$P:{\R}^{n+1}\to {\R}^n\iff P(z,t) = \frac{z}{t}$,dom $P={\R}^n×{\R}_{++}$

      透视函数对向量进行伸缩或规范化——将其最后一维分量置为1并舍弃

      具有保凸性：

      • 若$C\in$ dom P是凸的，其象$P(C)=\{P(x)|x\in C\}$也是凸的
      • 若$C\in {\R}^n$是凸的，其逆象$P^{-1}(C)=\{(x,t)\in {\R}^{n+1}|\frac{x}{t}\in C, t \ge 0\}$

      小孔成像原理——通过小孔看凸的物体还是凸的

    • 线性分式函数：

      函数$g：{\R}^n\to{\R}^{m+1}$是仿射的

      ​ $g(x) = \left[ \begin{matrix} A \ c^T \end{matrix} \right]x + \left[ \begin{matrix} b\ d \end{matrix} \right]$

      ​ $A\in {\R}^{m×n},b\in{\R}^m,c\in{\R}^n,d\in{\R}$,

​ 函数$f = P\circ g,f:{\R}^n\to {\R}^m\iff$ $f(x)=(Ax+b)/(c^Tx+d)$, dom $f = {x|c^Tx+d>0}$，成为线性分式（或投射）函数

由透视函数和仿射函数复合而成

若$c=0,d>0$，则$f$的定义域为${\R}^n$，$f$是仿射函数——仿射和线性函数为特殊的线性分式函数

利用矩阵表示:

用Q左乘$(x,1)$,即可得到$(Ax+b,c^Tx+d)$,继而用透视函数$P$作用于之上,即可得到$(f(x),1)$,$f(x) = \frac{Ax+b}{c^Tx+d}$

 具有保凸性:
  • 若$C$是凸的且$\{c^Tx+d>0 ~for ~x \in C\}$,则$f(C)=\{\frac{Ax+b}{c^Tx+d}|x\in C\}$是凸的
  • 若$C\in {\R}^m是凸的,则其逆象f^{-1}(C)=\{x|\frac{Ax+b}{c^Tx+d}\in C\}$是凸的

广义不等式

  • 正常锥:锥$K\subset {\R^n}$为正常锥,若$K$满足以下条件

    • $K$是凸的
    • $K$是闭的
    • $K$是实心的,内部非空
    • $K$是尖的$\iff$单向的,不包含直线$\iff$$x\in K,-x\in K \Rightarrow x=0$

    利用锥$K$来定义广义不等式: $x \preceq_K y \iff y-x \in K$

    • 严格偏序关系: $x \prec_K y \iff y-x \in$ int K

    并不是任意两个元素都可比,存在两个元素之前不存在大小关系的情况

  • 非负象限和分量不等式:

    • 正常锥$K={\R}^n_+$
    • $x \preceq_K y \iff x_i \le y_i, i = 1,…,n$
    • $x \prec_K y \iff x_i < y_i, i = 1,…,n$

  • 半正定锥和矩阵不等式

    • 正常锥$K = {S}^n_+$
    • $X \preceq_K Y \iff Y-X ~is~PSD $
    • $X \prec_K Y \iff Y-X ~is~positive ~definite $

  • 广义不等式的性质

    • 加法保序性
    • 传递信
    • 非负数乘保序性
    • 自反性
    • 反对称性
    • 极限运算保序性

    严格广义不等式的性质:

    • 满足严格一定满足广义
    • 严格+广义$\to$严格
    • 严格+数乘($\alpha>0$)$\to$严格
    • $x \nprec_k x$
    • 若$x\prec_k y$,对于足够小的$u,v$有$x+u\prec_ky+v$

  • 最小元:$x\in S$ is the minimum element$\iff$ $\forall y \in S, 满足x\preceq_k y$

    $S\subset x+ K$

    若最小元存在,则其唯一

  • 极小元:$x\in S $ is a minimal element $\iff$ if $y\in S,y\preceq_K x$ only if $y = x$

    $(x-K)\cap S = \{x\}$

    极小元可能有多个

  • 同理可定义极大元和最大元

[Optimization Methods]——Notes 1

数学补充——分析

  • 内部：对于$x\in C\subset \R^n$,若存在$\epsilon>0$满足:$\{y|~||y-x||_2 \le \epsilon\}\subset C$,称$x$为$C$的内点，$C$的所有内点组成$C$的内部，记为int C
  • 开集：若int C = C,则称C为开集
  • 闭集：若$C\subset \R^n$的补集$R^n$ $C$ = $\{x\in\R|x\notin C\}$是开集，则称$C$为闭集
  • 闭包：集合$C$的闭包定义为：cl C = $\R^n$ int($\R^n$ C)——$C$的补集的内部的补集

点$x$属于$C$的条件为：对与每一个$\epsilon>0, \exist y \in C$满足$||x-y||_2 \le \epsilon$

利用收敛序列和极限点的观点：

  • 集合$C$是闭集$\iff$它包含它的每个收敛序列的极限点
  • $C$的闭包是$C$的所有收敛序列的极限点组成的集合
  • 边界：bd C = cl C int C

边界点$x$的性质：

  • 对所有$\epsilon > 0$,存在$y\in C$和$z\notin C$满足$||y-x||_2\le\epsilon,||z-x||_2\le \epsilon$

集合$C$是闭集的条件为：bd C $\subset C$

集合$C$是开集的条件为：$C \bigcap$ bd C = $\emptyset$

仿射集合

  • 若通过集合C中的任意两点的直线仍然在集合C中，则称集合C是仿射的。

仿射组合：$\theta_1+…+\theta_k = 1$,$\theta_1x_1+…+\theta_kx_k$称为$x_1…x_k$的仿射组合。

线性方程组的解集是一个仿射集合，$C=\{x|Ax=b\}$,与仿射集合C相关联的子空间就是A的零空间

  • 零空间：算子$A$的零空间是$Av = 0$的所有解$v$的集合
  • 子空间：C是仿射集合且$x_0\in C$,则集合$V=C-x_0=\{x-x_0|x\in C\}$,是一个子空间，关于加法和数乘封闭。

同理：仿射集合C可以表示为：$C=V+x_0=\{v+x_0|v\in V\}$

  • 仿射包：集合$C\subset R^n$中的点的所有仿射组合组成的集合称为$C$的仿射包，即为aff C

aff C = $\{\theta_1x_1+…+\theta_kx_k|x_1,…,x_k \in C, \theta_1+…+\theta_k=1\}$

  • 仿射包是包含$C$的最小仿射集合。
  • 相对内部：集合$C$的相对内部为它相对于aff C的内部，记为relint C，即：relint C = $\{x\in C|B(x,r)\bigcap$aff C $\subset C$ 对于某些$r>0\}$
  • 相对边界：cl C relint C——集合$C$的闭包除去集合$C$的相对边界

凸集

  • 凸集：集合$C$被称为凸集$\iff\forall x_1,x_2\in C$和满足$0\le \theta\le1$的$\theta$都有$\theta x_1+(1-\theta)x_2 \in C$

仿射集合一定是凸集

凸组合：$\theta_1x_1+…+\theta_kx_k$,为$x_1,…,x_k$的一个凸组合，其中$\theta_1+…+\theta_k=1$且$\theta_i \ge 0, i = 1,…,k$

一个集合是凸集$\iff$集合包含其中所有点的凸组合

  • 凸包：集合$C$中所有的的凸组合的集合称为其凸包，记为conv C，

conv C = $\{\theta_1x_1+…+\theta_kx_k|x_i\in C,\theta_i\ge0, i = 1,…,k,\theta_1+…+\theta_k = 1\}$

凸包conv C总是凸的，它是包含$C$的最小凸集

锥

  • 锥：若对任意$x\in C$和$\theta \ge 0$, 都有$\theta x \in C$,则将集合$C$称为锥或非负其次。
  • 凸锥：若$C$是锥且$C$是凸的，则称$C$为凸锥$\iff$ $\forall x_1,x_2\in C和\theta_1,\theta_2 \ge0,有\theta_1x_1+\theta_2x_2 \in C$

锥组合（非负线性组合）：$\theta_1x_1+…+\theta_kx_k,\theta_1,…,\theta_k \ge 0$形式的点称为$x_1,…,x_k$的锥组合

集合$C$是凸锥$\iff$它包含其元素的所有锥组合

  • 锥包：集合$C$的所有锥组合的集合：$\{\theta_1x_1+…+\theta_kx_k | x_i \in C, \theta_i \ge 0, i = 1,…,k\}$

集合$C$的锥包是集合$C$的最小的凸锥

凸集的例子

  • $\emptyset$、单点集$\{x_0\}$、全空间$\R^n$
  • 直线是仿射的，通过零点的直线是凸锥
  • 线段是凸的但不是仿射的
  • 射线：$\{x_0+\theta v|\theta \ge 0\}$ ，$v\ne 0$ 的集合是凸的，但不仿射；若$v=0$,则其是凸锥
  • 任意子空间是仿射的、凸锥

  • 群：一个集合和定义在该集合上的操作的组合若满足下列条件则称为群

    • 在该操作上的封闭性

    • 操作的结合律

    • 集合中存在不变元

    • 集合中存在逆元

      若额外满足操作的交换律则称为Abelian Group

  • 向量空间：一个集合、定义在集合内部元素之间的操作和与定义在集合中元素与外部集合元素之间的操作组合（e.g. 内部：加法；外部：数乘）

    • 除去外部操作的得到的群是Abelian Group

    • 对于外部操作满足分配律和结合律

    • 对于外部操作也存在不变元

  • 向量子空间：$V = (\cal V,+,·)$是向量空间，集合$\cal U \subset \cal V$且$\cal U \ne \emptyset$，$U=(\cal U,+,·)$称为$V$的向量子空间,记为$U\subset V$,如果以下条件满足：

    • $U$关于$\cal U ×\cal U$、$\R × \cal U$封闭
    • $0 \in \cal U$

  • 关于子空间和向量空间的一些性质：

    • 对任一向量空间$V$，其自身$V$和$\{0\}$都是其子空间
    • 子空间至少要包含零元($0$)

超平面和半空间

  • 超平面：具有如下形式的集合$\{x|a^Tx=b\}$,$a\in\R^n,a\ne 0,b\in \R$ $\iff$ $\{x|a^T(x-x_0)=0\} = x_0+a^{\bot}$,$x_0$为超平面上任意一点

$a^{\bot}$表示$a$的正交补，即与$a$正交的向量的集合：$a^{\bot}=\{v|a^Tv=0\}$

超平面由偏移$x_0$加上所有正交于法向量$a$的向量构成。

  • 半空间：一个超平面将$\R^n$划分为两个半空间：$\{x|a^Tx\le b\},a\ne 0\iff \{x|a^T(x-x_0)\le 0\}$,$x_0$是超平面上的任意一点。

半空间是凸的但不是仿射的

  • 若半空间为$\{x|a^Tx\le b\}$
    • 半空间的边界——超平面$\{x|a^Tx=b\}$
    • 半空间的内部——开半空间$\{x|a^Tx<b\}$

球和椭球

  • 球：

    $B(x_c,r)=\{x|~||x-x_c||_2\le r^2\} = \{x|(x-x_c)^T(x-x_c)\le r^2\} = \{x_c+ru~|~||u||_2\le 1\}$

$向量x_c$是球心，$标量r$是半径，$B(x_c,r)$表示所有距离$x_c$不超过$r$的所有点的集合

球是凸集

  • 椭球：

​ $\varepsilon = \{x|(x-x_c)^TP^{-1}(x-x_c)\le1\} = \{x_c+Au~|~||u||_2 \le 1\}$

$x_c\in \R^n$是椭球中心，$P=P^T\succ 0$,即$P$是对称正定矩阵——决定了椭球从$x_c$向各个方向的拓展幅度,半轴长为$\sqrt\lambda_i$

$\varepsilon = \{x_c+Au~|~||u||_2 \le 1\}$，其中可以假设$A=P^{\frac{1}{2}}$是对称正定的，当$A$为对称半正定矩阵，但奇异时，集合称为退化的椭球，仿射维数为$A$的秩。

退化的椭球也是凸的

范数球和范数锥

  • 范数球：$\{x~|~||x-x_c|| \le r\}$

    范数球是凸的

  • 范数锥：$C=\{(x,t)~|~||x||\le t $ \} $\subset \R^{n+1}$

    范数锥是一个凸锥

    • 二阶锥：

    $C=\{(x,t)\in \R^{n+1}|~||x||_2 \le t$}$\iff x^2-t^2 \le 0, t \ge 0$

多面体和单纯形

  • 多面体：$\cal P = \{x|a_j^Tx\le b_j\, j = 1,…,m,c_j^Tx=d_j,j=1,…,p\}$

    多面体对应有限个线性不等式和等式的解集

    一系列半空间和超平面的交集

    • 矩阵表示：

      $\cal P = \{x|Ax \preceq b, Cx = d\}$

    • $u \preceq v \iff \forall i,~ u_i \preceq v_i $

  • 单纯形：设$k+1$个点$v_0,….,v_k\in \R^n$仿射独立，那么这些点决定了一个单纯形：

    C = conv $\{v_0,…,v_k\}$=$\{\theta_0v_0+…+\theta_kv_k~|~\theta \succeq 0, 1^T\theta = 1\}$

    • 仿射独立：$k+1$个点$v_0,…,v_k\in \R^n$仿射独立，即，$v_1-v_0,…,v_k-v_0$线性独立

    单纯形是一类重要的多面体。

    • 上述单纯形的仿射维数为$k$,也成为$\R^n$空间的$k$维单纯形。

  • 常见的单纯形：

    • 一维单纯形——线段

    • 二维单纯形——三角形及其内部

    • 三维单纯形——四面体

Patterns in backward flow

add gate: gradient distributor

max gate: gradient route
  • The branch whose input data is maximum will get the upstream gradient while the other branches will get the gradient of 0.
mul gate: gradient switcher
  • The local gradient is just the value of the other variable.

数字图像处理（第二讲：数字图像基础）

视觉感知要素

人眼

结构

  • 虹膜：控制进入眼的光量
  • 视网膜：物体在视网膜上成像
  • 晶状体：光线通过晶状体进入眼睛，晶状体可调节焦距
  • 中央凹(concavity)：视网膜上对颜色/细节敏感的区域

特点

  • 成像焦距：14-17mm

  • 亮度适应：

  • 人眼可感知的光强度范围很广——暗视觉和亮视觉两种不同感知细胞的感光范围所构成的感光范围很广

  • 人眼可以同时分辨的强度范围有限——给定条件下，仅有一定的感知范围，超出的部分是无法区分颜色的

    人眼的错觉和欺骗性

  • 马赫带效应：亮度条带交界处发生灰度的上冲和下冲

  • 与不同亮度对比：与不同亮度对比后，人眼对同一亮度的感知有所不同

  • 视错觉：平行，长短

    光和电磁波谱

    电磁波谱

  • $波长\lambda = \frac{光速c}{频率\nu}$

  • $能量E=普朗克常数h×频率\nu$

    可感知的光

    物体的颜色：

  • 物体均匀反射：呈白色

  • 反射特定光：呈该颜色

    单色光：

  • 没有颜色的光

  • 唯一属性：强度

  • 对单色光的感知：黑色（最暗）、灰色、白色（最亮）

  • 灰度：表示强度的数值

  • 单色图即灰度图像

    彩色光：

  • 范围：0.43-0.79mm的电磁波

  • 辐射（radiance）：能量（瓦特）
  • 光强（luminance）：感知的能量（流明）

  • 亮度（brightness）：主观描绘

  • 观测精度:波长需要小于物体尺寸才可观察

    图像感知和获取

    图像产生

    传感器

  • 将照射能量变化为数字图像

  • 分类：

    • 单个传感器：光二极管
    • 条带传感器：CT、核磁共振
    • 阵列传感器：数码相机、摄像机

    图像形成模型

  • 二维函数$f(x,y)=i(x,y)r(x,y):0<f(x,y)<\infty$

  • 入射分量：$0<i(x,y)<\infty$——表征入射光强度
  • 反射分量：$0<r(x,y)<1$——表征反射率

  • 投射系数：表征光穿透物体的程度

    灰度范围

  • 单色图像的灰度值：$L_{min} \le l \le L_{max}$

    ​ 最小值：$L_{min}=i_{min}r_{min}$

    ​ 最大值：$L_{max}=i_{max}r_{max}$

  • 灰度范围：

    • [$L_{min},L_{max}$]
    • 归一化：[0, L-1]
    • 0表示黑色，$L-1$表示白色，其余为灰色

图像取样和量化

基本概念

  • 目的：把连续图像转换为数字图像
  • 取样：对坐标进行数字化
  • 量化：对幅值进行数字化

表示数字图像

  • 二维数组$f(x,y)$:

    二维数组中每个元素的值为像素

    • 包含$M$行，$N$列
    • $x$=0,1,2,…,$M$-1
    • $y$=0,1,2,…,$N$-1
    • $f(x,y)$表示在$(x,y)$处的值

  • 空间域

    • 图像坐标所张成的实平面

  • 空间坐标

    • $x$和$y$

  • 灰度级数$L=2^k,k\in Z$

    • 灰度值$[0,L-1]$内的整数

  • 动态范围：

    • $\frac{最大可检测灰度}{最小可检测灰度}$

  • 图像的对比度：

    • 图像内最高和最低灰度之间的差值
    • 动态范围光、对比度高

存储数字图像

  • 存储数字图像的比特数:$b=M×N×k$
  • $k$比特图像：图像有$2^k$个灰度级别时，称为$k$比特图像

分辨率

  • 空间分辨率：

    • 图像中可辨别的最小细节的度量
    • 单位距离线对数
    • 单位距离点数
      • 单位英尺点数（dpi）
    • 空间单位
      • 图像像素大小并不能表示清晰程度
      • 像素大小可反映设备成像能力
    • 空间分辨率过低：图片失真

  • 灰度分辨率

    • 灰度级别中可体现的最小变化
      • 灰度级别通常是2的整数次幂
    • 用比特数表示灰度分辨率
      • 256个灰度级别意味着8比特分辨率
    • 比特数过低，灰度分辨率过低：出现伪轮廓

  • 空间、灰度分辨率的联合影响：

    • 等偏爱曲线：

      • 沿等偏爱曲线，人对画质的感受不变
      • 图像所需细节越多，需要的比特数越少（伪轮廓被大量细节掩盖）
      • 增加空间分辨率，可适当降低比特数

最佳量化

  • 使量化误差最小的量化方法

    • 使用均方误差量化质量

  • 计算过程

    • $Z、q$分别代表图像灰度和其量化值
    • $p(Z)$为像素灰度概率密度函数
    • $Z$的取值范围为$[H_1,H_2]$, 量化层数为$K$
    • 均方误差

    即将图像真实灰度为$[Z_k,Z_{k+1}]$的点，用离散量化值$q_k$代替，$p(Z)$事先已知，求$Z_i,i\in[i,K+1],q_i,i\in[1,K]$,令均方误差最小
